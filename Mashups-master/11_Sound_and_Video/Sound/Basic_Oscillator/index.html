Oscillator Example

Wave Type: sine

    Playback Controls
  • ENTER - press to play sound
  • SPACEBAR - press to stop sound
  • UP ARROW - press during playback to increase volume
  • DOWN ARROW - press during playback to decrease volume
  • LEFT ARROW - press during playback to decrease the wave frequency
  • RIGHT ARROW - press during playback to increase the wave frequency
    Wave Type Controls (Stop playing before changing)
  • 0 - set wave type to 'sine'
  • 1 - set wave type to 'square'
  • 2 - set wave type to 'sawtooth'
  • 3 - set wave type to 'triangle'
